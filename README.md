# 📝 Task Manager

A simple full-stack task manager app to create, update, and delete tasks.

## 📺 Live Demo

Live Link: [task-manager-vscs.onrender.com](https://task-manager-vscs.onrender.com)

## ⚙️ Tech Stack

**Frontend** — React, Axios  
**Backend** — Node.js, Express  
**Database** — MongoDB (Mongoose)  
**Deployed on** — Render

## 💽 Features

- Create a task with a title and content
- Update an existing task via a modal
- Delete a task

## Getting Started

### Prerequisites
- Node.js
- MongoDB connection string

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Environment Variables

Create a `.env` file in your backend folder:

```
MONGO_URI=your_mongodb_connection_string
PORT=3000
```
